### Bug found: security-scoped resource guard

`openSlideshow()` and `addImages()` used `guard url.startAccessingSecurityScopedResource() else { return }` which bails on accessible but non-scoped URLs (temp dirs, some file-importer URLs). This was likely the root cause of the "Add Images" picker not working. Fixed to use `let didStartAccessing = ...` pattern.
